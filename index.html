WDT Editor

World Definition Table editor for the online video game World of Warcraft.

The program takes a binary file known as a World Definition Table (.wdt) and enables you to edit different flags and chunks in the file.

This project was mainly used to learn and understand how to edit a file that did not contain traditional text, only Hexadecimal values. I did this by using the BinaryReader and BinaryWriter filestream features
